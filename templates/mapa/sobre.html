O Projeto

R.O.T.A.

Registro e Observação de Trajetórias de Aatualizações. Uma plataforma digital dedicada a centralizar e mapear casos de desaparecimento.

Acreditamos que a tecnologia pode ser uma ferramenta poderosa para conectar pessoas e mobilizar comunidades. A plataforma R.O.T.A. foi criada com o objetivo de fornecer um recurso centralizado, visual e de fácil acesso para auxiliar na divulgação e localização de pessoas desaparecidas.

Este projeto utiliza dados georreferenciados para transformar registos de desaparecimento em pontos visuais num mapa interativo, permitindo que qualquer pessoa possa consultar os casos de forma rápida e intuitiva.

Como Funciona

Cadastro

Utilizadores autorizados podem registar novos casos, incluindo fotos, descrições e a última localização conhecida.

Mapeamento

O sistema converte endereços em coordenadas e exibe todos os casos num mapa interativo, com marcadores e popups.

Informação

A barra de pesquisa com sugestões e os relatórios estatísticos ajudam a filtrar e analisar os dados rapidamente.

Tecnologias Utilizadas

Python Django Django Rest Framework JavaScript Leaflet.js TailwindCSS SQLite
